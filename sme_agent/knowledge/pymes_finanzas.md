# Fundamentos financieros para PyMEs

## Flujo de caja
- Separa ingresos operativos de ingresos extraordinarios.
- Establece politicas de cobranza claras y seguimiento semanal.
- Negocia plazos con proveedores para alinear pagos con cobros.

## Rentabilidad y costos
- Define costos fijos y variables y revisa margenes por producto o servicio.
- Ajusta precios segun costos totales, propuesta de valor y competencia.
- Controla costos indirectos con presupuestos mensuales.

## Endeudamiento responsable
- Evalua capacidad de pago antes de tomar creditos.
- Compara tasa efectiva, plazo, garantias y comisiones.
- Evita financiar gastos corrientes con deuda de largo plazo.

## Presupuesto y proyecciones
- Usa escenarios (conservador, base, optimista) para ventas y gastos.
- Define metas trimestrales y revisa desviaciones cada mes.

## Indicadores clave
- Liquidez: razon corriente y prueba acida.
- Rentabilidad: margen bruto, margen neto, ROA.
- Endeudamiento: deuda/EBITDA, cobertura de intereses.

## Gestion de riesgos
- Diversifica clientes para reducir concentracion.
- Mantiene reservas de liquidez para contingencias.
- Asegura procesos criticos (inventario, cartera, pagos).
